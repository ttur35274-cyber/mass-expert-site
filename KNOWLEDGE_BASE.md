## 1. ОПИСАНИЕ ПРОДУКТА

### 1.1 Назначение

Счетчики-расходомеры массовые **Mass-EXPERT** предназначены для измерений:

| Параметр | Среда |
|----------|-------|
| Массовый расход и масса | Жидкость (в т.ч. нефть), газ |
| Объёмный расход и объём | Жидкость (в т.ч. нефть) |
| Объёмный расход газа, приведённый к стандартным условиям | Газ |
| Плотность | Жидкость |
| Температура | Рабочая среда |
| Концентрация компонентов | Двухкомпонентные жидкие смеси |

### 1.2 Принцип работы (Эффект Кориолиса)

Основан на измерении сил Кориолиса, возникающих при движении измеряемой среды по трубке:

1. **Массовый расход**: Измерительная трубка постоянно вибрирует. При отсутствии потока вибрация симметрична. Поток среды создаёт фазовый сдвиг, пропорциональный массовому расходу.
2. **Плотность**: Рассчитывается по собственной частоте колебаний трубок.
3. **Объёмный расход**: Вычисляется автоматически из массового расхода и плотности.
4. **Температура**: Измеряется встроенным термометром сопротивления.

**Состав:** Сенсор (измерительная камера с трубками + термометр сопротивления) + Трансмиттер (обработка сигналов, индикация, выходы).

## 2. КЛЮЧЕВЫЕ ПРЕИМУЩЕСТВА

| # | Преимущество | Описание |
|---|-------------|----------|
| 1 | **Прямое измерение массового расхода** | Не требует пересчёта объёма в массу. Исключает ошибки от изменений температуры, давления, плотности |
| 2 | **Высокая точность** | Жидкости: до ±0,1% / Газы: до ±0,5% |
| 3 | **Широкий диапазон измерений** | До 100:1 (для U/K/Δ — до 1000:1) |
| 4 | **Регулируемое демпфирование** | Настраиваемое время сглаживания показаний |
| 5 | **Долгий срок службы** | 20 лет. Нет подвижных частей — минимум обслуживания |
| 6 | **Многофункциональность** | Один прибор = расход + плотность + температура |
| 7 | **Встроенная самодиагностика** | Автоматическое сообщение о неисправностях |

## 8. ИЗОТЕРМИЧЕСКИЙ КОЖУХ (ОПЦИЯ 19S)

Теплоизолирующая оболочка для сенсора с подачей теплоносителя (пар, горячее масло).

**Назначение:** Поддержание стабильной температуры сред, критичных к перепадам.

**Проблемы решаемые кожухом:**

| Среда | Проблема | Без кожуха | С кожухом |
|-------|----------|------------|-----------|
| Битум | Вязкость ×10 при −10 °C | Погрешность до 15% | Точность ±0,1% |
| Сера | Нелинейная вязкость (120→160 °C — в сотни раз) | Нестабильные показания | Измерение при 135–145 °C |
| Парафины | Кристаллизация ниже точки помутнения | Забивание трубок | Нет отложений |
| Мазуты/Гудроны | Застывание при остановах | Длительный прогрев | Мгновенный перезапуск |

**Характеристики:**
- Температура: −196 … +350 °C
- Теплоноситель: пар, горячее масло, вода
- Давление теплоносителя: до 16 МПа
- Материал: 304 / 316L нержавеющая сталь
- DN08–DN250
- Подходит для типов U, K, W, L, D
